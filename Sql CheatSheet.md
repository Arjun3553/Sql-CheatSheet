# SQL Database

## Create Database

```
create database databasename;
```

## List All Databases

```
show databases;
```

## Current Database

```
select database();
```

## To Use A Database

```
use databasename;
```

## Deleting Database

```
drop database databasename;
```

## Deleting Database If It Exists

```
drop database if exists databasename;
```
